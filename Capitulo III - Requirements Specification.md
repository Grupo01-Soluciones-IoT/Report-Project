## 3.2. User Stories

### Épicas

| **Epic ID** | **Título**                                   | **Descripción**                                                                                                                                                                 |
|-------------|----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1           | Desarrollo del Dispositivo IoT                                  | Desarrollar un dispositivo IoT que permita la recolección y transmisión de datos agrícolas en tiempo real. Incluye el diseño del hardware y desarrollo del firmware. |
| 2           | Desarrollo de la Aplicación Móvil                               | Desarrollar una aplicación móvil, asi como visualizar datos en tiempo real, recibir notificaciones y generar reportes.                      |
| 3           | Integración de Datos y Sistema de Comunicación                  | Implementar el sistema de comunicación entre el dispositivo IoT y la aplicación móvil para asegurar la transferencia eficiente y precisa de datos.                                |
| 4           | Implementación de Funcionalidades de Análisis y Recomendaciones | Desarrollar funcionalidades en la aplicación para analizar datos y proporcionar recomendaciones para la gestión de cultivos, utilizando algoritmos.                          |
| 5           | Desarrollo de Reportes y Herramientas de Visualización          | Crear herramientas para generar reportes y visualizar datos históricos y actuales, incluyendo gráficos y tablas en diferentes formatos.                                       |
| 6           | Pruebas y Validación del Sistema                                | Realizar pruebas exhaustivas del sistema completo, incluyendo pruebas de integración, usabilidad, resistencia del hardware y validación de datos.                               |
